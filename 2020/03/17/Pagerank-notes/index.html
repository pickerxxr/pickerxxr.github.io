简介

  • PageRank算法：谷歌推出，制胜法宝。
  • 其实每一条边都没有确定的数值大小，只能说是相对大小的赋值。

算法相关

理想情况（简单的情况）

  • A, B, C分别表示三个网页，任意一个箭头代表从此网页可以跳转到另一个网页。

  • 网页X的重要性记为I(X)，则有

$$
I(A) = \partial g(x) I(C)2 \
\ I(B) = I(A)/2 + I(B)/2 \ \
I(C) = \partial g(x) I(A)2 + \partial g(x) I(B) 1
$$

矩阵迭代的方法

  • initialize 一个 以每个节点出度的矩阵，即每个节点指向其他节点的指向性概率。
  • 比如以上的实例图中，就可以建立矩阵：[0, 1, 1; 0, 0, 1; 1, 1, 0]，这里用到的是概率相等的指向，实际上是由其相对可能性来决定的，当然可以将这个矩阵单位化。
