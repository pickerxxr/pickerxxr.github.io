    • ORDER BY语句：排序输出一个查询结果，其中ASC语句是升序，DESC指的是降序

    • INTO子句：非常简单但是十分有效的一个语句，作用是把查询结果导入到一个新的表格当中。

    • UNION语句：能够将两个或者更多查询结果合并为单个结果集，该结果集包含联合查询中的所有查询的全部行

    • 使用COMPUTE和COMPUTE BY子句汇总

      • 基本形式：
      • 各选项的含义如下：
      • AVG：数值表达式中所有值的平均值
      • COUNT：选定的行数
      • MAX：最大值
      • MIN：最小值
      • STDEV：标准偏差
      • STDEVP：总体标准偏差
      • SUM：和
      • VAR：方差
      • VARP：总体方差
      • 注意事项：
      1. DINSTINCT语句不能与集合函数一起使用
      2. COMPUTE语句中指定的列必须是SELECT子句中现有的
      3. 因为COMPUTE子句产生的是非标准行，所以COMPUTE子句不能与SELECT INTO语句一起使用
      4. COMPUTE BY 必须与ORDER BY一起使用，且二者指定的列相同或者前者是后者的子集，两者从左到右的顺序也必须相同。
      5. 不能再COMPUTE或COMPUTE BY 子句中指定ntext、text或image数据类型。
